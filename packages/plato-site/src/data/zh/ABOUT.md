# 关于柏拉图 (About Plato)

> *“他不仅是最善良的人，也是最智慧的人。”*

**柏拉图（Plato）是一个伪装成解谜游戏的证明助手。** 它的名字来源于那位坚信“真知源于推导，而非被动告知”的哲学家。

我做这个项目，是因为受够了逻辑学在课本里枯燥死板的模样。市面上大多数证明助手（比如 Lean 或 Coq）都硬核得令人望而生畏：它们操着晦涩的语法，甩给你几百页的硬核文档，默认你已经拿到了数学博士学位。

而柏拉图想换个好玩的方式。当你输入 `(assume A)`，看着新的一行证明蹦出来，接着敲下几行命令——不知不觉间，你其实已经在用蕴含式进行严丝合缝的思考了。

它不打算教你逻辑学的**概念**，它只想让你真正去**玩**逻辑。毕竟，最好的学习方法永远是亲自动手、疯狂踩坑，然后慢慢培养出对证明的肌肉记忆。

---

## 怎么玩？

游戏开始时，你会看到一个待证明的逻辑命题。你的武器是一个基于 S-表达式的 REPL 终端[cite: 1]：
* 输入指令——比如 `(assume A)`、`(->-elim 2 1)`、`(and-intro 1 3)`[cite: 1]
* 核心引擎会根据自然演绎规则，逐行死磕你的推导是否严密[cite: 1]。

别慌，前几关会有**引导卡片**手把手带你熟悉套路，卡住时还有**提示系统**捞你一把[cite: 1]。每解出一道题，你的“技能树”（战术面板）就会解锁新的推理规则，一步步充实你的工具箱[cite: 1]。

目前关卡里塞了 **45 道精心挑选的题目**[cite: 1]：
* 从最基础的身份律（只要三步，两分钟通关）[cite: 1]
* 到大名鼎鼎的皮尔士律、排中律和德摩根定律[cite: 1]。
* 甚至还有关于“苏格拉底是凡人”的谓词逻辑推导[cite: 1]。

如果你觉得这些还不过瘾，游戏还支持**自定义题目加载**（通过 URL 或 JSON 文件）——只要你敢写，引擎就敢验[cite: 1]。

## 为什么偏偏是“自然演绎”？

因为自然演绎模仿的，是人类真实大脑的推理路径。

大多数逻辑课最喜欢考真值表。画个格子一行一行填，一列一列查。它确实死板有效，但那感觉纯粹是在机械搬砖，不像是在*思考*。

自然演绎完全不同：你做出假设、推导出结论、最后消去假设。每一步都对应着你脑子里的真实直觉——*“如果我知道 p，那我就一定能拿到 q。”* **你不是在计算真假，你是在构建坚不可摧的论据。**

---

## 友情提示：它可能有点掉头发

丑话说在前面，自然演绎真的挺难。构建证明没有一劳永逸的通关公式，你必须得自己去找“手感”。

有时候，看起来只有一行的题目能把你卡到怀疑人生。卡关了极其正常，别硬顶，放它一马。去洗个澡或者睡一觉，第二天回来瞅一眼，往往就秒解了。**这不是一个能让你十分钟速通的新手教程，而是一种需要日积月累的脑力修行。**

## 幕后黑科技 (Tech Stack)

Plato 的底层是一个典型的 pnpm monorepo[cite: 1]：

* **硬核大脑 (`plato-lib`)**：纯手工打造的 Rust 形式化验证引擎，通过 `wasm-pack` 编译为 WebAssembly 注入前端[cite: 1]。公式、上下文、断言、推理规则、S-表达式解析器全都在这，大概 3000 行极度严谨的 Rust[cite: 1]。
* **轻量骨架 (`plato-site`)**：基于 Vue 3 + TypeScript + Vite 的单页应用[cite: 1]。为了保持极简和极速，我甚至**连路由器（vue-router）都没装**，全靠一个优雅的 TypeScript 判别联合类型和过渡动画来切页面[cite: 1]。
* **颜值担当**：数学公式交给 KaTeX 实时渲染，其余一切文本全部使用我最爱的程序员御用字体 **JetBrains Mono**[cite: 1]。
* **语言无障碍**：完整的中英文双语支持（i18n），甚至为每一个硬核逻辑学术语都精心编写了双语术语表[cite: 1]。
